# Trilha Diária — TRT • Fase 1 (Analista Judiciário, Área Administrativa)

> **Como este arquivo é lido pelo app**
> O aplicativo lê este arquivo no primeiro uso e gera automaticamente as
> matérias, tópicos e tarefas do módulo **Plano de estudos**.
>
> **Formato de cada tarefa** (uma por linha, começando com `- `):
> `Matéria | Tópico | Descrição | Recurso (link, opcional) | Meta de questões (opcional)`
>
> - Use `—` ou deixe vazio quando não houver link de recurso.
> - A meta de questões é um número (pode ficar vazia).
> - Cada semana começa com `## Semana N — Título`.
> - Cada dia começa com `### Dia N` (o complemento após `—` é só descritivo).
>
> Você pode editar livremente este arquivo e reimportá-lo pela tela **Dados**.
